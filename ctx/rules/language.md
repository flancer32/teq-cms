# Project Language Policy

Path: ctx/rules/language.md

- **All documentation** (including architecture, product descriptions, specifications and organizational texts) is written **in English**. English is the sole working language of the project.
- **User interface** is designed **in English**. All UI texts, labels and explanatory messages use English as the only language.
- **Comments in code, logs, error messages and service strings** are written **in English**, matching the broader technical ecosystem.
- Any deviation from this language policy is considered an error and must be corrected.
